# frontend-to-backend

## Setup
```
npm init -y
npm install express
npm i --save-dev nodemon
npm run dev
```

## Notes
* Browser > Network > Check "Disable cache"
* If showing cannot get info, run npm dev again
* in postInfo, remember to add (e) in postInfo
